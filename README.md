## 安装使用

- 获取项目代码

```
git clone git@github.com:zhangzy56/V-Admin.git
```

- 安装依赖
```
cd V-Admin
pnpm install
```

- 运行
```
pnpm run dev
```

- 打包

```bash
pnpm run build:pro
```
